# ChainKit

多鏈 SDK，目前開發中的鏈：Tron

## 技術棧

- .NET 10.0
- C#（Nullable enabled, ImplicitUsings enabled）
- Solution: ChainKit.slnx

## 專案結構

- `Tron/` — Tron SDK 類別庫

## 指令

- Build: `dotnet build`
- Test: `dotnet test`

## 慣例

- 語言：C#
- 命名：PascalCase（public）、_camelCase（private field）
- 所有 public API 需有 XML doc comment

## 文件

- `docs/decisions/` — 架構決策紀錄（ADR）
- `docs/` — 業務邏輯與技術文件
